# Claude Code Tutorial

A comprehensive guide to becoming a proficient Claude Code user.

## What is Claude Code?

Claude Code is Anthropic's official CLI tool that brings AI-powered assistance directly into your terminal. It helps with software engineering tasks including writing code, debugging, refactoring, and navigating codebases.

## Who is This For?

- Developers new to Claude Code who want to learn the fundamentals
- Intermediate users looking to improve their workflow
- Teams onboarding new members to Claude Code

## Quick Start

1. Install Claude Code:
   ```bash
   npm install -g @anthropic-ai/claude-code
   ```

2. Launch in your project directory:
   ```bash
   cd your-project
   claude
   ```

3. Start with [Chapter 1](chapters/01-getting-started.md) to learn the basics.

## Prerequisites

- Node.js 18 or higher
- Basic familiarity with the command line
- A code project to practice with

## How to Use This Tutorial

Each chapter builds on the previous one. For best results:

1. Read chapters in order
2. Try the examples in your own terminal
3. Complete the exercises at the end of each chapter
4. Practice on real projects
